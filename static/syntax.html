Compiler syntax:

Introduction:

The compiler syntax was made to be very easily to parse and program.

Because of that, preprocessing in the official compiler is not supported.

Basic syntax:

Every label is defined as:

LABEL_NAME:

NOTE: because of compiler issues, DO NOT put code after the colon, otherwise the compiler will ignore your code, and label name CANNOT be spaced.

Every mneoic is defined as:

INSTRUCTION ARGUMENT...,
